# Address Search App

A simple web application that allows users to search for addresses using the French government's address API. The app provides a clean, modern interface with support for pagination and Google Maps integration.

## Features

- Search addresses using the French government's [address API](https://adresse.data.gouv.fr/outils/api-doc/adresse).
- Real-time pagination for large result sets.
- Clickable links and icons to view addresses directly on Google Maps.
- Validation to ensure proper input format (e.g., addresses must start with a number).
- Modern, responsive design for an improved user experience.

## Technologies

### Frontend
- **HTML5**: Structured content and forms.
- **CSS3**: Custom styles for a clean and modern look.
- **JavaScript (Vanilla)**: Client-side functionality and API integration.

### Backend
- **Python**: Server-side programming.
- **Flask**: Lightweight web framework.
- **Requests**: For calling the French address API.

## Prerequisites

Before you begin, ensure you have the following installed on your machine:

- Git ([Download Git](https://git-scm.com/))
